But since you're here, it would be a great place too learn about HTTP Response Codes, Wikipedia has something about it.

They are a code that the server sends back when you, the client, requests something. The 400s are client errors, which means you did something wrong. 404 is probably the most common, and it basically means that whatever you are trying to find doesn't exist. other examples are 504 Gateway Timeout, 200 Ok, and 418 I'm a teapot.
